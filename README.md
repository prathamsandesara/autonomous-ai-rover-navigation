
Where:
- `g(n)` is the cumulative terrain cost from the start
- `h(n)` is the heuristic estimate to the goal

## âš¡ Battery-Aware Planning

- The rover continuously monitors its battery level
- If the goal is unreachable with current battery:
  - A* is re-run toward reachable recharge stations
  - The station closest to the final goal is selected
- Recharge behavior overrides normal goal planning

---

## âš ï¸ Hazard-Aware Navigation

- Discovered hazards are stored in memory
- Trap and hazardous tiles become **dynamic obstacles**
- Immediate backtracking is triggered when danger is detected
- Known hazards are skipped during node expansion

This allows the rover to **learn from past mistakes**.
